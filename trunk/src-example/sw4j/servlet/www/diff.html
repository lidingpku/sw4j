Diff Web Service

Features:
  • diff two RDF documents
  • REST web service interface
Process Diff Request
cur URL:
prev URL:

REST Service Interface Description

Base service URI:  http://onto.rpi.edu/sw4j/diff
parameter meaning
cur the URL of current RDF document
prev the URL of previous RDF document
example usage
  • simple diff (default) - http://onto.rpi.edu/sw4j/diff?cur=http://data-gov.tw.rpi.edu/catalog/2009/07-03-today.rss&prev=http://data-gov.tw.rpi.edu/catalog/2009/06-24-today.rss
  • simple diff (inverse) - http://onto.rpi.edu/sw4j/diff?prev=http://data-gov.tw.rpi.edu/catalog/2009/07-03-today.rss&cur=http://data-gov.tw.rpi.edu/catalog/2009/06-24-today.rss
