Convert Web Service

Features:
  • convert an RDF document into desired syntax
  • REST web service interface
Process Convert Request
URL:
Option: check if it uses valid RDF syntax
Option: output syntax

REST Service Interface Description

Base service URI:  http://onto.rpi.edu/sw4j/ping
parameter meaning
url the URL to be ping
view output rdf syntax :  RDF/XML, N3, N-TRIPLE, RDF/XML-ABBREVE .
Default output: N3
example usage
  • simple convert (default N3) - http://onto.rpi.edu/sw4j/convert?url=http://www.cs.rpi.edu/~dingl/foaf.rdf
  • simple convert (return N-TRIPLE) - http://onto.rpi.edu/sw4j/convert?url=http://www.cs.rpi.edu/~dingl/foaf.rdf&view=N-TRIPLE
