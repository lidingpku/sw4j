SW4J web services

  • SPARQL Service - (i) create a new RDF graph by executing a "DESCRIBE" or "CONSTRUCT" SPARQL query. (ii) "SELECT" query with OWL DL inference support.
  • PING Service - ping a web page.
  • Diff Service - diff two RDF documents (cur - prev).
  • Convert Service - convert an RDF into a different format.
  • Normalization Service - normalize an RDF graph.
note: this is an open source project, and the source code can be found at http://code.google.com/p/sw4j/
