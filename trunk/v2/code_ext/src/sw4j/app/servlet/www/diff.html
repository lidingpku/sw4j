SemDiff Web Service

Features:
  • diff two RDF documents
  • REST web service interface
Process Diff Request
cur URL:
prev URL:
output:
xmlbase of output:
rss_title:

REST Service Interface Description

Base service URI:  http://onto.rpi.edu/sw4j/diff
parameter meaning
cur the URL of current RDF document
prev the URL of previous RDF document
output the output format, any of the following ('rss', 'rdf_add', 'rdf_del')
rss_title required when output=rss, title of rss channel
rss_url required when output=rss, url of rss channel
example usage
  • simple diff (default) - http://onto.rpi.edu/sw4j/diff?cur=http://data-gov.tw.rpi.edu/raw/92/2009-07-10/today.rss&prev=http://data-gov.tw.rpi.edu/raw/92/2009-07-09/today.rss
  • simple diff (inverse) - http://onto.rpi.edu/sw4j/diff?prev=http://data-gov.tw.rpi.edu/raw/92/2009-07-10/today.rss&cur=http://data-gov.tw.rpi.edu/raw/92/2009-07-09/today.rss
  • simple diff (rss) - http://onto.rpi.edu/sw4j/diff?cur=http://data-gov.tw.rpi.edu/raw/92/2009-07-10/today.rss&prev=http://data-gov.tw.rpi.edu/raw/92/2009-07-09/today.rss&output=rss&xmlbase=http://data-gov.tw.rpi.edu/catalog/diff.rss&rss_title=Updates%20of%20data.gov%20catalog
