  • decouple recursive list structure
  • materialize transitive relation
  • compute deductive closure (RDFS)

REST Service Interface Description

Base service URI:  http://onto.rpi.edu/sw4j/norm
parameter meaning
url the URL of an RDF document
type_uri (only for unsign option) the URIs (separated by comma) of the class to be unsigned
tp_uri (only for tp option) the URI of the transitive property
example usage
  • simple sign (default) - http://onto.rpi.edu/sw4j/sing?uri=http://inference-web.org/proofs/tptp/Solutions/PUZ/PUZ001-1/EP---1.0/answer.owl
