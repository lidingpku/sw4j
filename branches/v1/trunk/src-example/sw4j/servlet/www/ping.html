Ping Web Service

Features:
  • ping a web page
  • optionally check its RDF syntax
  • REST web service interface
Process Ping Request
URL:
Option: check if it uses valid RDF syntax
Option: output syntax

REST Service Interface Description

Base service URI:  http://onto.rpi.edu/sw4j/ping
parameter meaning
url the URL to be ping
view (optional) output rdf syntax :  RDF/XML, N3 .
Default output:  RDF/XML for describe/construction;  TEXT/PLAIN for other queries
vrdf (optional) validate RDF syntax
example usage
  • simple PING (default) - http://onto.rpi.edu/sw4j/ping?url=http://tw.rpi.edu/
  • simple PING (return RDF) - http://onto.rpi.edu/sw4j/ping?url=http://tw.rpi.edu/&view=RDF/XML
  • PING + RDF validation - http://onto.rpi.edu/sw4j/ping?url=http://www.cs.rpi.edu/~dingl/foaf.rdf&vrdf=yes
